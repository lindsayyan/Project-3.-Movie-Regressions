Can machine learning be used to predict the box office perfomances of movies.

Techniques Used:

Data Probing

Data from this project was taken from a Kaggle Moie Database Initially we searched the movie metadata for clues to this issue, following EXTENSIVE feature engineering of course. Any trends noted were recorded in a dashboard.

NLP and Random Forest Regressor

Natural Language Processing was used to dissect the movie descriptions. The intent was to build a model that not only determines if there are trendy words or word combinations to determine whether movies will be hits but also will provide and estimate of Revenue that movie will earn at the box office. A Random Forest Regressor was used to build the final model.

Inputs: Movie description, Budget

Outputs: Revenue



Ensemble Analysis for Movie Revenue Classification

Next we created an Regression analysis using common movie data. Initially this model was created using Random Forest but the resulting model was a poor fit based on the scores. After examining the results of the intial run, we use RFE to eliminate features and built a new model using ensemble. The ensemble model was more accurate when we used it to classify the movie as into two categories, those with above average revenue potetial and those below. we examined the movie metadata and reduced the features

Inputs: Genre, Released date, Budget, Ratings, Popularity

Outputs: Movie Revenue Classification based on an average of USD $117,000,000
