Ensemble Model



Variables Used:

Before running the ensemble model, we used RFE (Recursive Feature Elimination) to do variable selection, then we ran a simple Logistic regression to see the performance on the Logit model with only selected variables, it turned out that all the variables that were selected using RFE has a P value less than 0.05, meaning the variables are signifant to the outcome varible. and the model has an R square of 36%. It seems like these variables have a good prediction power for the outcome variable.

The Outcome variable is binary, 1 means the movie revene is above average USD$117,000,000 (Good), 0 means the movie revenue is below average (Bad). The reason why we are predicting a binary outcome instead of continous is because we found the variables does not describe the outcome variable well (Revenue of Movie). Then we decided to make the problem into a classification problem and the model showed better results than the linear regression model. Therefore we decided to change the model into a classification model





Model Used For Ensemble:

Ensemble model is a Logistic Regression model running on top of a series models that were trained to predict the outcome variable. The Models that we used for the Ensemble model are:

Support Vector Machine (SVM)

K Nearest Neighbors (KNN)

Naive Bayes (NB)

Multilayer Perceptron Neural Network (MLPNN)

Random Forest (RF)

Gradient Boosting (GBM)

Logistic Regression (LR)

After plotting out the ROC curve for all Models including Ensemble, it is clear that the ensemble model is much better than all the other models that were trained. The Esemble model has a ROC of 83%, whereas all the other models have ROC around 70%

F1 Scores:

Ensemble model also has the highest F1 Score compare to each individual models at 82%, individually GBM is the best performer at 81%, NB is the worst at 73%. The scores here are for the test dataset.
